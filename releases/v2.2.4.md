# Cloudflare KV Manager v2.2.4

## Highlights

- **Dependency Updates** — Bumped `@cloudflare/workers-types`, `@types/node`, `jose`, `typescript-eslint`, `wrangler`
- **Security Patches** — Updated `tar` override 7.5.10 → 7.5.11 (npm CLI + Docker layers)
- **GitHub Actions** — Bumped Docker actions to latest majors (`setup-buildx` v4, `login` v4, `metadata` v6, `build-push` v7)
- **Build Fix** — Removed empty `vendor-react` manual chunk (Vite 7 automatic JSX runtime)
