# v2.2.5

This patch release resolves high-severity vulnerabilities in our dependency tree and bumps major tools, including Vite, now at major version 8.

### Security

- **undici vulnerabilities**: Addressed multiple high-severity vulnerabilities in `undici`, overriding its exact version, fixing:
  - GHSA-f269-vfmq-vjvj / GHSA-2mjp-6q6p-2qxm / GHSA-4992-7rv2-5pvq
  - GHSA-vrm6-8vpv-qv8q / GHSA-v9p9-hfj2-hcw8 / GHSA-phc3-fgpg-7m6h

### Changed

**Dependency Updates**

- Bumped `vite` to `^8.0.0` and `@vitejs/plugin-react` to `^6.0.1` and adapted `manualChunks` configuration
- Updated generic project dependencies to their latest compatible ranges via `npm update`
